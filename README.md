# Proiect JDBC Persoane și Excursii

## Descriere
Acesta este un proiect Java care utilizează **JDBC** pentru a interacționa cu o bază de date **MySQL**. Proiectul permite gestionarea persoanelor și a excursiilor lor, cu opțiuni precum adăugare, afișare și ștergere.

## Funcționalități
1. Adăugarea unei persoane (nume și vârstă).  
2. Adăugarea unei excursii pentru o persoană existentă.  
3. Afișarea tuturor persoanelor și a excursiilor lor.  
4. Afișarea excursiilor pentru o persoană după nume.  
5. Afișarea persoanelor care au vizitat o anumită destinație.  
6. Afișarea persoanelor care au făcut excursii într-un anumit an.  
7. Ștergerea unei excursii.  
8. Ștergerea unei persoane împreună cu toate excursiile ei.

## Tehnologii folosite
- Java 17+  
- JDBC  
- MySQL  
- Maven  
- IntelliJ IDEA
